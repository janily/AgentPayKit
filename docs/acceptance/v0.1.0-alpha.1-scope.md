# AgentPayKit v0.1.0-alpha.1 release scope

Status: Developer Preview only.

This release freezes a community preview with one supported payment path:

- Base Sepolia only;
- fixed USDC pricing per Endpoint;
- synchronous JSON request/response;
- manual wallet confirmation for every paid call.

Not included in this release:

- Base Mainnet support;
- dynamic, token, or time-based pricing;
- subscriptions;
- asynchronous jobs or long-running tasks;
- automatic payment retries or autonomous wallet signing;
- marketplace or no-code publishing flows.

The three publishable packages in this scope are:

- `@agentpaykit/server`
- `@agentpaykit/cli`
- `create-agentpay-skill`

All three packages target the `0.1.0-alpha.1` pre-release line.
